FUNCIONAMENTO DO PROJETO PARA DESENVOLVIMENTO:

Passo 1 > Instalar servidor/webserver PHP v5.x;

Passo 2 > Instalar Git v2.x:
			Link > https://git-scm.com/downloads

Passo 3 > Clonar Repositório qisat/player;

Passo 4 > Baixar e instalar Node JS v8.x:
			Link > https://nodejs.org/en/download/

Passo 5 > Instalar Gulp e os plugins necessários pela Linha de Comando:
			1. npm install --save-dev gulp gulp-mode gulp-iconfont gulp-iconfont-css gulp.spritesmith merge-stream gulp-sass gulp-rename typescript gulp-typescript gulp-concat gulp-watch 

Passo 6 > Editar host e virtual host: _root/dist

Passo 7 > Rodar a tarefa "gulp" na Linha de Comando;

Pronto! O projeto está pronto para uso do desenvolvedor.
